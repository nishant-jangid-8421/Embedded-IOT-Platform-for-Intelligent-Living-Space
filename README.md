# Embedded IoT Platform for Intelligent Living Space

🏆 **First Prize Winner – Spectrum 2K26**
📄 **Research Paper Published – ICETIS 2026**

## Overview

The **Embedded IoT Platform for Intelligent Living Space** is a distributed ESP32-based smart living environment designed to enhance safety, security, automation, energy efficiency, and remote monitoring.

The system utilizes multiple ESP32 nodes to monitor environmental conditions, perform intelligent local decision-making, and provide real-time remote access through IoT connectivity.
